Код для слияния двух списков двумя способами
1 способ:
a-создаем список
b-создаем второй список
c=a+b—объединяет списки a и b создаётся новый список c
print(c)-выводим список c

2 способ:
a-создаем список
b-создаем второй список
a.extend(b)-этот код используется для слияния списков
добавляет все элементы списка b в список a
print(a)-выводим новый список a
